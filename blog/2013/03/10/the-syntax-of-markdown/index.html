The Syntax of Markdown

Mar 10th, 2013

Octopress使用的是markdown语法，和wiki的语法有点相似，但又不完全相同，这里将常用的语法记录下来便于查阅。

标题

标题用#开头，一个#是一级标题，两个#是二级标题，#越多字体越小。

换行

在每一行的末尾以两个或更多个空格符号结束，然后再打回车键，即可实现 <br />的效果。

链接

[text](link) 创建文本链接。

图片

与链接类似，区别是前面增加叹号!：

![text](link)

引用

用>开头的一个段落。

用四个空格缩进的段落，会按原始格式显示。也可以通过在引用段落的前后各加一行波浪号~~~~~~来实现。

显示效果

用*或_包围的文字会用斜体显示。

双重符号**或__则会用粗体显示。

用`包围的文字按代码格式显示。

列表

无序列表用*, +, -开头。子项缩进两个空格。

有序列表用数字加英文句点。

水平线

在一行里只放三个或更多个\，或*或_，就可以实现水平线标记<hr />

脚注

以[^1]: 开头，后面跟着定义。

嵌入代码

 ``` [language] [title] [url] [link text]
   code snippet
 ```

转义

用\可转义Markdown元字符。
